Resources Inventory
===================

The resources inventory contains the list of the used components in a setup. In the :guilabel:`Rack
Resources Usage` panel, you can select the rack you wish to consult. In a single rack system the rack is
automatically selected, in a multi-rack setup, you have to click the name of the rack.

When you have selected a rack, you can see the various devices in the rack. Click on any of the resources
to consult its inventory. The selected resource is indicated in blue in the rack.
